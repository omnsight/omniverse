## 📦 Installation Guide

This project is managed with [uv](https://github.com/astral-sh/uv).

Install/Upgrade dependencies:

```bash
npm install
```

Clean up:

```bash
npm run clean
```

## 🚀 Run Service Locally & Debug

Run the application:

```bash
npm run dev
```

## ✅ Running Unit Tests

```bash
npm test
```

## 💅 Formatting Code

Format the code using black:

```bash
npm run lint
```

## Deploy To Production

```bash
npm run build
```
