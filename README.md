# Nexuse Intelligence Platform

### 🚀 Features

- 🗺️ Interactive map and graph visualizations for data exploration
- 📝 A rich text editor for creating and editing content
- 🪟 A flexible window management system for arranging and organizing different views
- 🌐 Support for multiple languages (English and Chinese)
- 🔐 Secure authentication using OpenID Connect (OIDC)

### 🛠 Tech Stack

- [React](https://react.dev/) - A JavaScript library for building user interfaces
- [Vite](https://vitejs.dev/) - A fast build tool for modern web development
- [TypeScript](https://www.typescriptlang.org/) - A typed superset of JavaScript
- [Mantine](https://mantine.dev/) - A full-featured React components library
- [Zustand](https://zustand-demo.pmnd.rs/) - A small, fast, and scalable state-management solution
- [React Query](https://tanstack.com/query/latest) - A data-fetching and state-management library
- [Reactflow](https://reactflow.dev/) - A library for building node-based graphs
- [Leaflet](https://leafletjs.com/) - An open-source JavaScript library for mobile-friendly interactive maps

## 📦 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/en/) (v20.15.1 or higher)
- [npm](https://www.npmjs.com/) (v10.7.0 or higher)

### Installation

Clone the repo:

```bash
git clone https://github.com/omnsight/omniverse.git
cd omniverse
```

Install dependencies:

```bash
npm install
```
