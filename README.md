# Omniverse

Omniverse 是一个基于 React 和 TypeScript 构建的情报数据可视化平台，专注于地理空间情报展示和管理。该平台集成了多个微服务，提供事件可视化、数据分析和管理功能。

## 快速开始

### 启动前端开发服务器

```bash
npm install
npm run dev
```

前端将在 http://localhost:5173 上运行。

### 构建生产版本

```bash
npm run build
```

构建的静态文件将位于 `dist` 目录中。

## 调试命令z

### 前端调试

```bash
# 运行 TypeScript 类型检查
npx tsc --noEmit

# 运行 ESLint 代码检查
npm run lint

# 运行所有测试
npm test
```
